# Next.js Project Generation Spec

## Objective

Create a production-ready Next.js application using the latest stable version of Next.js with TypeScript, App Router, Tailwind CSS, and a clean scalable folder structure.

### Tech Stack

- Next.js (latest)
- TypeScript
- Tailwind CSS
- App Router
- ESLint
- Prettier
- React Query (TanStack Query)
- Axios
- Zod
- React Hook Form
- shadcn/ui

### Instructions

1. You have to create a UI of an online coding platform like leetcode / hackerrank
2. The first component consists of the heading of the problem and below it consists of the problem description
3. The second component consits of the sample input and sample outputs where the they have to be enclosed inside of a code like background
4. The third component cosists of the code editor itself , use a mininal library of your choice to acheive the same . below the code editor there has to be to buttons called run and submit.
5. The second compoenent should be below the first one and together they should be on the left and the third one should be on the right.
6. utilize shadcn components.
